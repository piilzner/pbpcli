# pbpcli
cli tool for installing pbp [boilerplates](https://github.com/piilzner/pbp)

## Installation
```
$ git clone https://github.com/piilzner/pbpcli.git
$ cd pbpcli
$ npm install
$ npm link
```

## Usage
```
$ pbp --help
$ pbp install <boilerplate> <directory name>
$ pbp list 
$ pbp generate <type> <file name> 
```
